How does this work???

  • cmx.io provides CSS and JavaScript
  • You provide the HTML

HTML

Provides STRUCTURE and CONTENT 

<h1>A list of things!</h1>
<ul>
  <li>First this</li>
  <li>Then this</li>
</ul>

CSS

Provides STYLING 

code {
  padding: 2px 4px;
  color: #d14;
  background-color: #f7f7f9;
  border: 1px solid #e1e1e8;
  white-space: nowrap;
}

JavaScript

Provides LOGIC 

$('#option').select(function() {
  alert('An option was selected!');
});
